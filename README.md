# 🌐 Browser Simulation

## 📌 Proje Hakkında

**Browser Simulation**, C# ve Windows Forms kullanılarak geliştirilmiş basit bir masaüstü uygulamasıdır.
Bu proje, bir web tarayıcısının bazı temel davranışlarını **simüle etmek** amacıyla geliştirilmiştir.

Projede özellikle **veri yapıları** kullanılarak şu işlemler gösterilmektedir:

* 📚 **Stack** → Tarayıcı geçmişi (History)
* 📥 **Queue** → Dosya indirme işlemleri

Bu sayede veri yapılarının gerçek bir uygulamada nasıl çalıştığı gösterilmektedir.

---

## 🚀 Özellikler

* 🌍 Web sitesi ziyaret etme simülasyonu
* 📚 **Stack kullanarak tarayıcı geçmişi saklama**
* 📥 **Queue kullanarak dosya indirme simülasyonu**
* ⏳ **ProgressBar ile indirme ilerleme göstergesi**
* 🪟 Yeni sayfanın ayrı bir formda açılması
* ✅ İndirme tamamlandığında kullanıcıya mesaj gösterilmesi

## ⚙️ Nasıl Çalışır

1️⃣ Kullanıcı bir site adı girer.
2️⃣ **Visit** butonuna basıldığında site yeni bir formda açılır.
3️⃣ Önceki sayfa **Stack veri yapısına** eklenir.
4️⃣ Kullanıcı bir dosya indiriyormuş gibi simülasyon başlatabilir.
5️⃣ İndirme isteği **Queue veri yapısına** eklenir.
6️⃣ **ProgressBar** indirme sürecini görsel olarak gösterir.
7️⃣ İndirme tamamlandığında dosya kuyruktan çıkarılır ve kullanıcıya bildirilir.
